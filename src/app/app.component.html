CUPCAKE CLICKER TUTORIAL

This is a step by step tutorial on creating a frontend and backend for a simple cupcake clicking game with users and a leaderboard.
Make sure to follow TDD for this project. If you are having issues some steps are given angular references.

BEGINNING TIPS

Debugging: When having issues try right clicking and choosing inspect. Here you can see errors in the console, or look into the network tab once you start making backend calls to see what data or errors are getting returned.

When adding new Modules, they will need to be added to the app.module.ts file, as well as your component and spec files that use them.
